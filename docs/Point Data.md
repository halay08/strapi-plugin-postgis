### Point Data Type Support

- POINT(0 0)

![Screenshot](https://github.com/halay08/strapi-plugin-postgis/raw/main/images/points.png?raw=true)

```json
"geom": {
      "columnType": {
        "type": "specificType",
        "args": [
          "geometry(POINT,4326)"
        ]
      },
      "type": "json",
      "fieldRenderer": "postgis"
    }

```
